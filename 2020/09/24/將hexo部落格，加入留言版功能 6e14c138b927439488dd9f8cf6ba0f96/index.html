將hexo部落格，加入留言版功能

發表於 2020-09-24 Disqus：

我選擇使用第三方服務「Disqus」

使用方法非常簡單

申請Disqus帳號與建立留言板

  1. 註冊Disqus帳號
  2. 登入後，點擊首頁的「GET STARTED」
  3. 選擇「I want to install Disqus on my site」
  4. 需要輸入的欄位
    • Website Name：自訂 short name，且是唯一，在設定 Hexo 時需要使用。
    • Category：類別，自選，我選Tech。
    • Language：語言，自行選擇，我選Chinese。
  5. 填寫後點擊「Create Site」
  6. 到 Settings 的地方，將部落格網址加入到 Website URL

設定Hexo

  1. 到部落格themes目錄之下的 _config.yml 檔案(我使用的theme是Next)，修改 disqus 區段的設定。

完成

此時會看到部落格文章已經加入留言板的功能。
